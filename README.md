# 🧾 Nota App

Aplikasi **Nota / Invoice sederhana** berbasis **HTML, Tailwind CSS, dan JavaScript**.
Project ini memungkinkan pengguna menambahkan daftar barang, menghitung total harga secara otomatis, serta menampilkan nota yang siap dicetak.

Project ini cocok untuk latihan dasar **Frontend Development**, terutama manipulasi **DOM JavaScript** dan styling menggunakan **Tailwind CSS**.

---

# 🚀 Demo Fitur

Fitur utama dari aplikasi ini:

* ➕ Menambahkan daftar barang
* 🧮 Menghitung total harga otomatis
* 🧾 Menampilkan tabel nota
* 🖨️ Cetak nota (print invoice)
* 📱 Responsive design
* 💾 Simpan data sementara dengan JavaScript

# ⚙️ Cara Menjalankan Project

### 1. Clone repository

```
git clone https://github.com/panggih1207/nota-app.git
```

### 2. Masuk ke folder project

```
cd nota-app
```

### 3. Jalankan project

Buka file `index.html` di browser.

Atau gunakan **Live Server** di VS Code agar lebih mudah saat development.

---

# 🧮 Cara Menggunakan Aplikasi

1. Masukkan **nama barang**
2. Masukkan **harga barang**
3. Masukkan **jumlah barang**
4. Klik tombol **Tambah**

Barang akan masuk ke dalam **tabel nota** dan total harga akan dihitung otomatis.

---

# 🖨️ Fitur Print Nota

User dapat mencetak nota dengan tombol **Print**.

Browser akan membuka dialog print sehingga nota bisa langsung dicetak seperti invoice.
